1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 가입한 유저의 수 N(1 ≤ N ≤ 100,000)
둘째 줄부터 N개의 줄에는 유저의 닉네임이 가입한 순서대로 한 줄에 하나씩


② 문제 유형 파악:
 접두사(Prefix) 중에서 가장 길이가 짧은 것을 사용
접두사가 이전에 가입한 닉네임의 접두사가 아니어야 한다
유저가 가입한 순서대로 별칭을 한 줄에 하나씩 출력

=> 트리 -> 트라이 사용
