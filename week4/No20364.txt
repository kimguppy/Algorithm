1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫 번째 줄에 땅 개수 N  (2 ≤ N < 220)
꽉꽉나라에 사는 오리 수 Q가 공백 ( 1 ≤ Q ≤ 200,000)
두 번째 줄부터 차례로 Q개의 줄에 걸쳐 i+1번째 줄에는 i번째 오리가 원하는 땅 번호 xi  (2 ≤ xi ≤ N)

② 문제 유형 파악:
오리들을 한 줄로 대기시킨다. 맨 처음 오리들은 1번 땅에 위치해 있다.
오리들이 서있는 순서대로 원하는 땅을 가지도록 한다. => 트리모양
Q개의 줄에 원하는 땅에 갈 수 있다면 0을, 갈 수 없다면 처음 마주치는 점유된 땅의 번호

=> 트리 사용

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ boolean[N + 1] 땅 개수를 담은 불리언 배열 생성 
➁ for(int root = ducks/2;  root >= 1; root= root/2){if(numArr[root]==true) havenot = root;}
ducks/2 => 가고자 하는 땅의 부모 노드, 목적지에서 루트까지 부모 노드 탐색 havenot: 가장 최근에 발견된 부모노드 저장
➂ if(havenot == 0) numArr[ducks] = true; 발견된 부모노드가 없을 시 0
