1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 테스트 케이스의 개수 ( N ≤ 100,000)
친구 관계가 생긴 순서대로

② 문제 유형 파악:
두 사람의 친구 네트워크에 몇 명이 있는지 구하는 프로그램을 작성
=> 유니온 파인드

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ int[] parent; // 유니온 파인드 루트 노드 배열
	int[] level; // 각 노드마다 층의 개수
➁ level이라는 1차원 배열을 정의하여 각 노드마다 층의 개수
➂  친구 관계에 해당하는 노드들을 합집합 연산
(y에 있던 층의 개수를 더해 줌.)
