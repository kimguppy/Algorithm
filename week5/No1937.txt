1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 대나무 숲의 크기 n(1 ≤ n ≤ 500)
둘째 줄부터 N개의 줄에 걸쳐 각 노드와 그의 왼쪽 자식 노드, 오른쪽 자식 노드


② 문제 유형 파악:
n × n의 크기의 대나무 숲
둘째 줄부터 n+1번째 줄까지 대나무 숲의 정보
첫째 줄에는 판다가 이동할 수 있는 칸의 수의 최댓값을 출력

=> 다이나믹 프로그래밍

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ int[][] map: 대나무 숲
int[][] dp: 판다가 살 수 있는 년도 표기
➁ dx = x + rangeX[i];
dy = y + rangeY[i]; => 상하좌우 검사 
➂ dp[x][y] = Math.max(dp[x][y], DFS(dx, dy) + 1); 상하좌우 중에서 가장 오랫동안 생존할 수 있는 기간을 계산
