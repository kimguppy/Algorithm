1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫 번째 줄에는 창문의 개수와 사람의 수 N(1 ≤ N ≤ 2,100,000,000)

➁ 문제 유형 파악:
N번째 사람은 N배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다. 
열려 있는 창문의 개수를 구하라. 단, 처음에 모든 창문은 닫혀 있다.
정수론 :: 배수

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ 입력 n => 크기 n의 배열 생성 nv
➁ for문 사용 N번째 사람을 뜻하는 i, 창문을 뜻하는 j 선언
=> J가 i의 배수일 시 배열 값 변경 (0->1, 1->0)
➂ for문으로 nv배열 값 1 갯수를 구함
