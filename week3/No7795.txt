1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 테스트 케이스의 개수 T
각 케이스의 첫째 줄에는 A의 수 N과 B의 수 M
둘째 줄에는 A의 크기가 모두 주어지며, 셋째 줄에는 B의 크기가 모두 주어진다. (1 ≤ N, M ≤ 20,000) 


② 문제 유형 파악:
 두 종류의 생명체 A와 B가 존재
 A는 B를 먹는다. A는 자기보다 크기가 작은 먹이만 먹을 수 있다.
 두 생명체 A와 B의 크기가 주어졌을 때, 
 A의 크기가 B보다 큰 쌍이 몇 개나 있는지 구하는 프로그램을 작성하시오.

=> 투포인터, 정렬

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ A와 B 입력 후 정렬
➁ (p < m && a[i] > b[p]) 각 케이스와 마다 비교 후 쌍의 개수 출력
