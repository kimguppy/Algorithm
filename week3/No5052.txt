1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 테스트 케이스의 개수 t가 주어진다. (1 ≤ t ≤ 50) 
테스트 케이스의 첫째 줄에는 전화번호의 수 n이 주어진다. (1 ≤ n ≤ 10000) 
다음 n개의 줄에는 목록에 포함되어 있는 전화번호가 하나씩
(전화번호의 길이는 길어야 10자리이며, 목록에 있는 두 전화번호가 같은 경우는 없다.)

② 문제 유형 파악:

주어진 전화번호 목록이 일관성이 있는지 없는지를 구하는 프로그램을 작성
한 번호가 다른 번호의 접두어인 경우가 없어야 한다.

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ number_list[] : 전화번호부 
➁ number_list의 앞에 값과 startsWith 함수를 사용해서 비교하여 boolean 값을 return
: startsWith() 함수는 대상 문자열이 특정 문자 또는 문자열로 시작하는지 체크하는 함수
