1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 N(1 ≤ N ≤ 1,000,000)

② 문제 유형 파악:
2×N 크기의 벽을 2×1, 1×2, 1×1 크기의 타일로 채우는 경우의 수를 구한 후
첫째 줄에 경우의 수를 1,000,000,007로 나눈 나머지를 출력

=> 다이나믹 프로그래밍 사용

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ dp = new long[N][2];
➁  N이 1일 때 경우의 수는 2, N이 2일 때 경우의 수는 7
N이 3이상일 때 N-1 크기에서 채우는 경우의 수는 2가지
N-2 크기에서는 채우는 경우의 수는 3가지
N-(3이상) 크기에서는 채우는 경우의 수는 2가지
➂ 2배를 더한 값을 저장하고 N번 째 경우의 수는 2배를 더한 값 + N-2의 경우의 수로 표현
