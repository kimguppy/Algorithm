1).해당 문제 해결 기법을 사용한 근거 (논리적인 근거)
① 문제인식
첫째 줄에 N (3 ≤ N ≤ 5000)

② 문제 유형 파악:
봉지는 3킬로그램 봉지와 5킬로그램 봉지가 있다.
상근이가 배달하는 봉지의 최소 개수를 출력한다. 
만약, 정확하게 N킬로그램을 만들 수 없다면 -1을 출력

=> 그리디 알고리즘 사용

2). 의사 코드 또는 DFA 다이어그램 작성 (대략적으로 작성)
➀ 입력받은 N 값 5 나누고 나머지가 없으면 바로 몫 출력
➁ 5로 나눈 나머지 3으로 나누기, 3으로 나눴을 때 나머지가 없으면 ➀의 목+➁의 몫 출력 
➂ 나머지가 있을 시, 5의 몫 하나 씩 줄인 후 ➁ 반복 값이 없을 시 -1 출력
